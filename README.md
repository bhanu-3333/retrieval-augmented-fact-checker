# Retrieval-Augmented Fact-Checker

This project is a retrieval-augmented fact-checker system.

## Project Structure

- **backend/**: Python-based backend service.
- **fake-news-frontend/**: Next.js-based frontend application.

## Getting Started

### Frontend
1. Navigate to `fake-news-frontend/`.
2. Run `npm install`.
3. Run `npm run dev`.

### Backend
1. Navigate to `backend/`.
2. Set up your virtual environment.
3. Install dependencies from `requirements.txt` (if available).
